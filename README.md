# Tailpipe Plugin SDK

This SDK makes it easy to write Tailpipee plugins.

* **[Go Documentation →](https://pkg.go.dev/github.com/turbot/tailpipe-plugin-sdk)**

## Compiling this SDK

To compile the SDK repo, you must install protoc.
<pre>
brew install protoc
brew install protoc-gen-go-grpc
</pre>
